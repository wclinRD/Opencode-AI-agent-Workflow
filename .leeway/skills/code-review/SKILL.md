# code-review

Code review checklist and guidelines.

## Review Checklist

1. **Correctness**
   - Logic is correct
   - Edge cases handled
   - No obvious bugs

2. **Security**
   - No security vulnerabilities
   - Input validation
   - No secrets in code

3. **Performance**
   - No obvious performance issues
   - Proper use of data structures
   - No unnecessary loops

4. **Style**
   - Follows coding standards
   - Clear naming
   - Proper documentation
